# Dashboard Tweaks
Makes the CiviCRM dashboard 1 column for better readbility without the need to use a horizontal scrollbar.

This is an [extension for CiviCRM](https://docs.civicrm.org/sysadmin/en/latest/customize/extensions/), licensed under [AGPL-3.0](LICENSE.txt).

## Getting Started
Enable 1 column layout under Adminster > System Settings > Display Preferences. 

Allows additional custom CSS to be applied.
